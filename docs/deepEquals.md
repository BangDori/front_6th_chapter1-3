깊은 비교를 통해 두 값이 같은지 비교하는 함수

```md
1. 참조가 동일한 경우 true 반환 (`===` 연산자를 활용하여 비교)
2. 비교하려는 값이 모두 객체 타입이며 null이 아닌 경우
    2-1. 객체의 키 개수가 다르면 false 반환
    2-2. 모든 키를 순회하며 값을 비교
        2-2-1. 값이 객체인 경우, 재귀적으로 값을 비교
        2-2-2. 값이 객체가 아닌 경우, `===` 연산자를 활용하여 비교 (값이 다르면 false 반환)
    2-3. 순회가 끝나면 true 반환
3. 객체가 아닌 경우, `===` 연산자를 활용하여 비교
```
